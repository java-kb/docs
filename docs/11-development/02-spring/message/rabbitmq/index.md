# Spring RabbitMQ
To use the AMQP and RabbitMQ features in Spring Boot applications.
```xml
<dependencies>
    <!-- ... existing dependencies -->
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-amqp</artifactId>
    </dependency>
</dependencies>
```
